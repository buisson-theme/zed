## Installation

**Via Zed marketplace**

`cmd+,` → Extensions → search "Buisson" → Install

**Manuellement**

```sh
git clone https://github.com/buisson-theme/buisson-zed.git && cp buisson-zed/themes/buisson.json ~/.config/zed/themes/
```

Puis `cmd+,` → Theme → Buisson Dark ou Buisson Light.

## Accessibility

- Primary text: **11.55:1 AAA** (dark) · **15.15:1 AAA** (light)
- All 6 accents: WCAG AA or above
- Calibrated against WCAG 2.1 — same profile as Flexoki.
